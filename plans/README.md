### 2026-06-24 — vetted and rejected (don't re-audit)

- **"XSS via incomplete HTML-entity decoding / `javascript:`-URL regex gaps"** —
  rejected. The render boundary is `<iframe sandbox="">` + injected `script-src
  'none'` CSP (`library-editor.tsx:270-274`, `htmlArtifacts.ts:3-6,64`); the
  validator is explicitly defense-in-depth. `decodeAttributeValue` is reached

- **"Version-number race / check-then-act in `updateArtifactWrite`"** — by-design.
  Convex mutations are serializable OCC transactions; concurrent writes conflict
  and retry.
- **"Version-mismatch on draft apply blocks legitimate edits"** — intentional
  optimistic concurrency (`libraryArtifactDrafts.ts:328`); a tradeoff, not a bug.
- **LCS diff O(n·m) in `library-artifact-draft-card.tsx`** — already mitigated by
  a 40K-cell cutoff (`:524`).
- **"Timeline re-renders on every message"** (ask-panel) — speculative, low impact;
  not planned.
- **Schema duplication between `artifacts` and `artifactDrafts`** — L effort, low
  value; not planned.
